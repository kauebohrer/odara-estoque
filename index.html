🔄 Como Recalcular a Projeção

O recálculo processa todos os materiais dia a dia e leva mais de 30 segundos — por isso não pode ser chamado diretamente pelo sistema. Siga os passos abaixo:

PASSO A PASSO
  1. Acesse o painel do Supabase
  2. Clique em SQL Editor no menu lateral
  3. Cole e execute o comando abaixo:
SELECT rebuild_saldo_estoque_diario(2026, 270);

Após a execução (30–60s), volte aqui e selecione o material para ver os dados atualizados.
